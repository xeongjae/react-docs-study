## 📚 스터디 진행방식
- 이해하기 어려운 개념을 토론하고, 책 외의 레퍼런스를 함께 참고하여 깊이 있는 이해를 추구합니다.
- 미팅 전까지 이해한 내용을 요약하고, 미팅에서 다룰 이야기를 정리하여 PR로 올립니다.
- 매주 화요일 오후 1시에 오프라인 미팅을 진행합니다.
- 각자 준비한 질문을 공유하고, 서로 답변을 시도해 봅니다.
- 매주 미팅 후, 다음 미팅까지의 목표 범위를 설정합니다.
